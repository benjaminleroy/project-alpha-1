## Raw `.tex` Files for the Report's Appendices

This directory contains the raw `.tex` files used to generate the content of the report's appendices. Please see the parent directory for instructions on how to generate the report. 

- `appendix_outliers.tex`: Examaines outlier removal using the RMSS technique from class assignments. 
- `appendix_smoothing.tex`: Additional notes on our smoothing method. 
- `appendix_convolution.tex`: A very detailed summary of all the steps and motivations leading up to our final version of convolution. 
- `appendix_clustering.tex`: Outlines a rejected clustering approach.
- `appendix_time.tex`: Discusses modeling voxel time courses as ARIMA processes.
